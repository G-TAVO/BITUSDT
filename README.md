# BITUSDT - Plataforma educativa de simulación

⚠ Proyecto de simulación  
No maneja dinero real. Solo para fines educativos y demostrativos.

## Funcionalidades
- Registro de usuarios desde cualquier dispositivo
- Login de usuarios y admin
- Panel de usuario con saldo, días y ganancias
- Botón de invertir (envía solicitudes al admin)
- Panel admin para aprobar solicitudes
- Retiro simulado
- Compartir link de referido

## Instalación
1. Clonar o descargar el repositorio
2. Ejecutar en la terminal:
```bash
npm install
npm start
